# navigation_astar

#### 介绍
基于图搜索的 A* 和 Jps 路径规划算法

#### 软件架构
软件架构说明


#### 安装教程

1.  mkdir -p ~/catkin_ws/src
2.  cd ~/catkin_ws/src
3.  catkin_init_workspace
4.  cd ~/catkin_ws/src
5.  git clone https://gitee.com/spiderman-spiderman/navigation_astar.git
6.  cd ~/catkin_ws/
7.  catkin_make
8.  source devel/setup.bash

#### 使用说明

1.  roslaunch grid random_map.launch
